# ✨ About the Project

**G45 Weekly Timetable Viewer** is a **luxury-styled CSS microproject** built by engineering students of **Group G45** to make academic scheduling look **modern, interactive, and examiner-friendly**.

Forget boring paper timetables or cluttered PDFs — this project delivers a **premium UI experience** with subject-wise color coding, smooth dark/light mode toggling, and a clean responsive layout that works across all devices.

> _"Your week, structured beautifully — because great engineers deserve great tools."_
---

# 🔥 Features

- 📅 **Weekly Grid Layout** — Clean, structured view of your entire week at a single glance.
- 🎨 **Subject-wise Color Coding** — Every subject gets a unique color for instant recognition.
- 🌗 **Dark / Light Mode Toggle** — Switch themes effortlessly. Easy on the eyes, day or night.
- 📱 **Fully Responsive** — Looks stunning on desktop, tablet, and mobile.
- ⚡ **Blazing Fast** — Powered by React + Vite for instant load times.
- 🏫 **Student Ready** — Built specifically for engineering students and academic presentations.

# 💻 Run Locally

### 1️⃣ Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/g45-timetable-viewer.git
```

### 2️⃣ Navigate into the project
```bash
cd g45-timetable-viewer
```

### 3️⃣ Install dependencies
```bash
npm install
```

### 4️⃣ Start development server
```bash
npm run dev
```

### 5️⃣ Open in your browser
```
http://localhost:5173
```
